1-"git clone aqui se coloca el enlace del repo a copiar " con este comando podemos traer un repositorio de github a nuestra git local.

3-"git commit -am "mensaje del commit" haciendo commit guardamos cambios,estos cambios se van a almacenar en la historia del repositorio.

4-"tags" con este comando podemos crear puntos de version en nuestro proyecto,es decir una version 1.0 o 2.0 etc.

5-"Pull reques" esta funcion es solo de github,sirve para resaltar un error y hacer el llamado a los colaboradores del repositorio,para debatir y corregir el error.
